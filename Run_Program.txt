How to Run The Program

1. Install Python
2. Setup Python
3. Clone the Repository
4. Open terminal and run `py -m venv src`
5. run `.\src\Scripts\activate.bat`
6. close the terminal
7. CTRL + SHIFT + P to change the python interpreter to virtual environment
5. In the terminal run `pip install -r requirements.txt`
6. run .\src\apps.py
